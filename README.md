# OBIS Landing Page (static)

This repository contains a small, copy-ready static landing page for OBIS, ready to deploy to Netlify.

## Quick start

1. Edit placeholders (YOUR_DOMAIN, social links, Buttondown/Chat IDs).
2. Push to GitHub and import the repo into Netlify (New site â†’ Import from Git).
3. Publish directory: root (`.`). No build command.
4. Submit test form after first deploy (Netlify detects forms automatically).

Files:
- index.html, thanks.html, privacy.html, 404.html
- netlify.toml, robots.txt, sitemap.xml
- assets/styles.css, images (placeholders)
